# JCBase

## Description
JCBase is a faster java version of PYCBase (which was written in python)
JCBase means Java Custom Base.

## Download
To download is => [here](https://github.com/Efendo/JCBase/zipball/main/)

## Why did i make it?
PYCBase is very slow. To generate all 11-Bit ternary numbers it took ~1.5 hours where as JCBase does it in 3.13 seconds
